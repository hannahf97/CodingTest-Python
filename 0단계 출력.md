## 대소문자 바꿔쓰기 

### 내가 만든 코드
```
str = input()
answer = ""
for i in str:
    if i.islower():
        answer += i.upper()
    else:
        answer += i.lower()
print(answer)

```python

### 다른사람이 쓴 코드
swapcase라는 함수 활용

```
print(input().swapcase())
```
join을 통해 하나로 합침

```
print(''.join(x.upper() if x == x.lower() else x.lower() for x in input())
```
